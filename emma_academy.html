EMMA Academy — System Guide

🔑 Accessing Instructor Mode

Click the "... by Francesco Richichi" credit text at the bottom of the landing page and enter the instructor password. Once logged in, these features become available:

  • Full access to the Track Builder
  • Assign Tracks buttons inside each scoring module
  • Save button to export the configuration files

🔧 Track Builder

The Track Builder creates the visual stereo field animation for each EMMA test track:

  1. Open Track Builder from the landing page.
  2. Build the stereo field animation (instrument positions, timing, audio).
  3. Click Save Track — a .json file is downloaded
    (e.g. 02 Position Left Overlay - EMMA 2026.json).
  4. Repeat for each track required.

📂 Assigning Tracks to Modules

Each scoring module has an Assign Tracks button (Instructor Mode only):

  • Position Score — tracks 02–06 (overlay, multiple instruments)
  • Focus Score — tracks 07–11 (single instrument)
  • Stage Score — track 12 (moving track)

Click Assign Tracks, select the .json files created with the Track Builder. The module loads them with animation and audio ready to play.

💾 Saving the Configuration

After assigning all tracks, click Save on the landing page. Three data files are downloaded (emma_data_position.json, emma_data_focus.json, emma_data_stage.json) and must be uploaded to the server alongside emma_academy.html.

🎓 Judge Workflow

Judges open the app URL, select a module, and press Play. No login required.
